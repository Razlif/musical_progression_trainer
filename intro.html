Welcome to the Ear Progression Trainer

Currently, the app supports diatonic chords / notes (only notes that are in the relevant major scale) and interval training:

  • You can choose between basic 3 note triads or 4 note 7th chords.
  • You can choose the length of the progression and add an opening or closing Tonic.
  • You can add a root base to the chord.
  • You can choose arpeggio or harmonic playback.
  • Practice identifying intervals with the Interval Trainer.

How to use the Chord Progression Trainer:

  1. Set the progression length to 1.
  2. Set the key to 'C'.
  3. Familiarize yourself with the different scale chord degrees.
  4. Gradually increase the length of the progression.
  5. Test yourself using random keys.
  6. Repeat the process using 7th chords.

Interval Trainer:

Improve your ability to identify musical intervals by training with the Interval Trainer.
